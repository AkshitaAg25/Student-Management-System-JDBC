# Student-Management-System-JDBC
A Java-based Student Management System that performs basic database operations using JDBC (Java Database Connectivity).
The application allows users to manage student records through a simple interface by interacting with a relational database.

This project demonstrates the use of Java, JDBC, and SQL to implement database connectivity and perform CRUD operations.

Features

Add new student records
View all student details
Update student information
Delete student records
Database connectivity using JDBC
Structured and modular Java code
